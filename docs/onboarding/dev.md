# Developer / AI Agent Onboarding

This guide is for a future human collaborator or a new AI agent joining SpeakFit.

## Read first

1. `BMAD_SPEAKFIT_WORKFLOW.md`
2. `IDEA_BRIEF.md`
3. `.cursor/rules/project-context.md`
4. `.cursor/rules/agent-routing.md`
5. `.cursor/rules/bmad-boundary.md`

## Operating model

- Solo PM is accountable for all product, scope, merge, and release decisions.
- BMAD agents are used for delivery workflows.
- Cursor advisory agents are explicit-only review gates.
- Every AI task should be framed as an AI Work Order.
- Every AI output should be submitted as an AI Artifact Contract.

## First contribution flow

1. Pick a Ready story or ask Solo PM for a work order.
2. Confirm input artifacts and acceptance criteria.
3. Work in a scoped branch if code changes are needed.
4. Submit artifact contract with verification evidence.
5. Wait for Solo PM approval.
